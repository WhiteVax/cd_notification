## cd_notification

cd_notification — это микросервис уведомлений, отвечающий за рассылку сообщений пользователям через различные каналы:

    Telegram-бот
    внутренняя система сообщений
    подписки на темы и категории

Сервис обеспечивает централизованную доставку уведомлений о событиях (интервью, отклики, утверждения, отказы, обновления) другим участникам платформы CheckDev. Он тесно интегрируется с другими микросервисами экосистемы — такими как cd_auth, cd_desc, cd_mock, cd_generator — и взаимодействует с ними через REST и Eureka.
